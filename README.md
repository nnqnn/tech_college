# Dating backend (проектирование)

Этот репозиторий содержит **описание сервисов**, **архитектуру** и **схему данных БД** для dating-приложения, в котором пользователь взаимодействует через **Telegram-бота**.

Документация:
- `docs/services.md` — сервисы и их ответственность
- `docs/architecture.md` — схема взаимодействия компонентов (Mermaid)
- `docs/database.md` — схема БД (Mermaid ERD) и ключевые правила хранения данных/рейтинга
